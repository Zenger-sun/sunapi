# Trademark and Brand Policy

The open source licenses in this repository grant rights to use, modify, and
redistribute covered code. They do not grant permission to impersonate official
SunAPI products, services, domains, release channels, or maintainers.

## Allowed

- Forking the code.
- Stating that a project is based on or compatible with SunAPI.
- Using the code under the applicable open source licenses.

## Not Allowed Without Permission

- Presenting a modified build as an official SunAPI release.
- Using SunAPI names, logos, icons, or domains in a way that confuses users
  about origin or endorsement.
- Removing required upstream attribution from AGPL-covered frontend code.

## Recommended Fork Naming

Use names like:

- `ExampleProxy, based on SunAPI`
- `ExampleProxy for SunAPI`
- `ExampleProxy, compatible with SunAPI`

Avoid names like:

- `Official SunAPI`
- `SunAPI Pro`
- `SunAPI Cloud`

unless you control the official release channel or have written permission.
